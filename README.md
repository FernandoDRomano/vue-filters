# Example filters with vue

This example uses vue, vuex, vue-paginate, vue-datepicker and tailwind.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
